# Chapter 4. Getting Started

我们提供了一个运行“spring scripts”的命令行工具。

我们的主要目标是：

* 为所有 Spring 开发提供一种速度极快且可广泛访问的入门体验。
* 开箱即用
* 提供一系列大型项目通用的非功能特性（例如嵌入式服务器、安全性、指标、健康检查和外部化配置）。
* 完全没有代码生成，也不需要 XML 配置。
